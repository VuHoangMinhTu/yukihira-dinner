Yukihira Dinner - Food Delivery & Reservation System

Giới thiệu chung

Yukihira Dinner là một nền tảng thương mại điện tử dành cho nhà hàng, cho phép người dùng đặt món trực tuyến và đặt bàn trước. Dự án tập trung vào trải nghiệm mượt mà, thanh toán an toàn và kiến trúc hệ thống hiện đại.

Công nghệ sử dụng

Frontend: React (Vite), TypeScript, Tailwind CSS (optional)

Backend: Java Spring Boot (REST API), Spring Security (JWT)

Database: PostgreSQL

Payment Gate: PayOS Integration

Những tính năng chính

Xác thực người dùng: Đăng ký, đăng nhập và phân quyền người dùng bảo mật với JWT.

Menu: Danh sách món ăn được cập nhật thời gian thực từ Backend.

Đặt món: Người dùng có thể chọn món, quản lý giỏ hàng và đặt đơn hàng giao về nhà.

Thanh toán: Tích hợp cổng thanh toán PayOS, đảm bảo quy trình giao dịch nhanh chóng và tin cậy.

Đặt bàn (Chưa hoàn thành bên phía Front-end): Tính năng đặt bàn trực tuyến đang trong quá trình hoàn thiện.
